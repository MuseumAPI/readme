Documentation for MLS's online presence.

How to edit this documentation:

  1. Clone the repository: 
    git clone git@github.com:lifeandscience/readme.git
  2. Install jekyll 
    sudo gem install jekyll
    Adding another line here!
  3. Edit the files, previewing your changes by running Jekyll from the terminal within the same folder: 
    jekyll --server
    • If editing CSS, use the Less command line or other automated tool (like CodeKit) to compile style.less into style.css
  4. Commit and push your changes! Pushed changes will be live at http://lifeandscience.cc/readme after a brief amount of time.
